Task 2: Intermediate HTML, CSS, and JavaScript

This project was completed as part of my Web Development Internship at ApexPlanet. The objective of this task was to improve my front-end development skills by building an interactive and responsive webpage using HTML, CSS, and JavaScript.

In this task, I created a contact form using HTML and CSS with input fields such as name, email, and message. The form was designed to provide a clean and user-friendly interface for users to enter their information.

To ensure proper data input, I implemented JavaScript form validation that checks whether the required fields are filled and verifies the correct email format before the form is submitted.

I also developed a responsive webpage layout using Flexbox and CSS Grid. Media queries were applied to make the webpage adapt to different screen sizes such as mobile, tablet, and desktop devices.

Additionally, I implemented JavaScript DOM manipulation by creating a dynamic to-do list, allowing users to add and remove tasks interactively. A dark mode and light mode toggle was also included to enhance the user experience.

This task helped me gain practical experience in responsive design, form validation, and interactive web functionality, strengthening my ability to build modern and dynamic web applications.
